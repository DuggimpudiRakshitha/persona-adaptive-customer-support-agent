# Persona-Adaptive Customer Support Agent

## Overview
An AI-powered support assistant that:
- Detects customer persona
- Retrieves relevant support documents
- Generates persona-adaptive responses
- Escalates sensitive issues

## Personas
- Technical Expert
- Frustrated User
- Business Executive

## Features
- Persona Classification
- Document Retrieval
- Adaptive Response Generation
- Human Escalation Workflow
- Streamlit Interface

## Installation

pip install -r requirements.txt

streamlit run app.py
